Python parameters

更新于 2016-05-27   |   分类于 python   |  

提纲

本文主要从以下几个方面分析python的参数机制：

  1. 固定参数(位置参数)
  2. 默认参数
  3. 可变参数
  4. 关键字参数

在每个部分中，我们区分函数的定义和函数的调用。
对于函数定义时的参数，我们称为parameter（形参）；对于函数调用时的参数，我们称之为argument(实参)。区分这两点对于解释清楚一些混淆的东西很重要

所以，以上的参数机制其实都是指的是函数定义时的形参，而不是调用时的实参！
