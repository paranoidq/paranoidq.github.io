Hexo博客管理神器：hexo hey

更新于 2016-05-19   |   分类于 hexo   |  

有了 hexo hey，妈妈再也不用担心我的hexo博客无法管理了。

安装

  1. install hexo-hey

  2. add admin user to _config.yaml

  3. hexo serve
    然后访问 http://localhost:4000/admin

注意问题

  1. 更新npm，否则有可能出现TypeError: Cannot read property '_id' of undefined]错误

  2. 关于添加多个tag或category的问题:
    需要手动Enter才能添加

参考
