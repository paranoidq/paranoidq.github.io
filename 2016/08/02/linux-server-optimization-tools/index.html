禁用反向域名解析，加快查询速度 -n

附：watch命令

watch可以帮助使用者监测一个命令的运行结果，避免重复手动运行。watch命令会周期执行
参数：

  • -n 时间间隔，缺省值为2s
  • -d 高亮显示变化区域
  • -t 关闭watch命令在顶部的时间间隔
